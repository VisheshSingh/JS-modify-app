A simple website-UI template built using Bootstrap 4 Beta

Project Description:

This is a single page website design of a simple user interface that I created using Bootstrap 4. It is a completely responsive piece of design built to be viewed on devices of various dimensions and layout.

For the workflow, I used Bootstrap 4 starter pack which uses Gulp (JS task runner) to compile all the Bootstrap source code, all of the SASS files, loads up the dev server and the dependencies.


The UI consists of a transparent navigation bar which is fixed at the top, then there is a showcase section for login. I used the grid system and flexbox for the general styling and pattern.


The rest of the UI consists of different sections, with stock images picked up from pexels.com. The footer section as you can see from the images above consists of the contact us button which when clicked slides down a contact form modal.


As previously mentioned, it is responsive on large devices like desktop PCs, tablets and even mobile phones; the navigation bar transorms into hamburger menu for smaller devices for giving a modern look and feel.

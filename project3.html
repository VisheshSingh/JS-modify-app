PropListings

A Property listings application built using Angular 2 and Firebase

Login: Firebase Auth

This is the first look of the application. For the front-end I used a customized version of Bootstrap called the Bootswatch. Users can login with their verified Google credentials which are basically handled by Firebase database at the back-end.

Home page

After the credentials are verified, users will be able to access their home page where they can see the listings for the properties or add a new property listing or even logout.

Property Listings

Once the user clicks on the listings tab, he/she will be brought to their property listings page where they can find the updated catalog of the properties on-sale within their account.

Property Details

Users can click on a specific property lisiting to view the detailed information about the owner, number of bedrooms, type and sale value of the property.

Add-listings

Users can also add their own property to the listing by click on the Add-Listing tab and filling in the form.
